# 📊 Data Analytics Project

## Overview

This project demonstrates an end-to-end Data Analytics workflow using Python, SQL, and Power BI. The objective is to transform raw data into meaningful insights through data cleaning, exploratory data analysis (EDA), SQL analysis, and interactive dashboard visualization. The project also includes a business report and presentation for communicating findings effectively.

## Dataset
The dataset contains business-related information used for data analysis and visualization.

The project involves:
* Loading the dataset into Python
* Exploring data structure and quality
* Cleaning and preparing the data
* Performing SQL-based analysis
* Building an interactive Power BI dashboard
* Creating a business report
* Designing a presentation using Gamma

## Project Workflow

### 1. Data Loading

* Import dataset into Python
* Inspect rows, columns, and data types

### 2. Exploratory Data Analysis (EDA)

* Analyze dataset structure
* Generate descriptive statistics
* Identify missing values
* Detect outliers
* Explore relationships between variables
* Create visualizations

### 3. Data Cleaning

* Handle missing values
* Remove duplicate records
* Correct inconsistent data
* Convert data types
* Rename columns
* Prepare data for analysis

### 4. SQL Analysis

Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server and perform SQL queries such as:

* Filtering data
* Aggregations
* GROUP BY
* ORDER BY
* JOIN operations
* CASE statements
* Common Table Expressions (CTEs)
* Window Functions
* Business KPI analysis

### 5. Power BI Dashboard

Create an interactive dashboard that includes:

* KPI Cards
* Bar Charts
* Line Charts
* Pie/Donut Charts
* Slicers and Filters
* Trend Analysis
* Business Insights

### 6. Report

Prepare a report summarizing:

* Data cleaning process
* EDA findings
* SQL insights
* Dashboard observations
* Business recommendations

### 7. Presentation

Create a professional presentation using Gamma covering:

* Project overview
* Dataset
* Methodology
* Key insights
* Dashboard screenshots
* Recommendations
* Conclusion

## How to Run
1. Clone this repository.
2. Install the required Python libraries.
3. Open the Jupyter Notebook files.
4. Run the notebooks in sequence:

   * Data Loading
   * EDA
   * Data Cleaning
5. Import the cleaned dataset into PostgreSQL.
6. Execute the SQL queries.
7. Open the Power BI dashboard (`.pbix`) to explore the visualizations.
8. Review the report and presentation for project insights.
